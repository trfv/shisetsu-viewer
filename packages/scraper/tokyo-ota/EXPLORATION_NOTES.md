## 基本情報

- **URL**: `https://www.yoyaku.city.ota.tokyo.jp/eshisetsu/menu/Welcome.cgi`
- **システム名**: うぐいすネット（大田区公共施設利用システム）
- **システムベンダー**: GrowOne（日成コミュニティ） — `eshisetsu` プラットフォーム
- **空き照会**: 利用者登録なしでも可能（「空き照会のみのかたはこちら」ボタン）

## GrowOne eshisetsu の一般的な特徴

- CGI ベース（`.cgi` URL）
- セッション管理あり
- 推定 URL パターン:
  - `Welcome.cgi` — トップ/ログイン画面
  - `Login.cgi` — ログイン処理
  - (推定) 施設分類選択 → 施設選択 → 室場選択 → 空き状況表示
- フレーム/iframe を使用している可能性あり
- 他の GrowOne インスタンス: `ibaraki-shisetsuyoyaku.growone.net`, `suita-shisetsuyoyaku.growone.net`

## フェーズ1で確認すべき事項

ローカル環境で `/new-scraper https://www.yoyaku.city.ota.tokyo.jp/eshisetsu/menu/Welcome.cgi tokyo-ota` を実行し、以下を確認:

1. **ナビゲーションパス**: トップ → 空き照会 → 施設分類選択 → 施設選択 → 室場選択 → 空き状況テーブル
2. **テーブル構造**: 行/列の配置、HTML セレクター
3. **時間区分テキスト**: テーブルヘッダーの生テキスト（"午前", "午後1" etc.）→ DIVISION_MAP
4. **ステータス記号**: テーブルセルの生テキスト/画像（"空き", "×" etc.）→ STATUS_MAP
5. **日付フォーマット**: 和暦/西暦、区切り文字
6. **ページネーション**: 次の日/週への遷移リンク/ボタン
7. **スタジオ vs 集会室**: 時間区分の違い（1コマ制 vs 午前午後夜間）

## テンプレート候補

サイト構造に応じて参照するテンプレート:
- **リンクチェーン + カレンダー + day-next** → `tokyo-chuo` or `tokyo-kita`
- **ドロップダウン + フォーム送信 + 「次へ」** → `tokyo-arakawa`
- **ポップアップ + 翌日リンク** → `tokyo-koutou`
